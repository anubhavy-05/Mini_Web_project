#  Mini Web Projects Collection

A comprehensive collection of web development projects and lectures showcasing HTML, CSS, and JavaScript skills.

##  Overview

This repository contains 14 projects and 6 lectures covering various web development concepts, from basic HTML/CSS to advanced JavaScript interactivity.

##  Live View Dashboard

Access all projects and lectures through our interactive dashboard:

**[Open Live View Dashboard](live-view.html)**

### Features:
-  **Responsive Grid Layout** - View all projects at once
-  **Smart Search** - Type a lecture/project name and press Enter for focused view
-  **Beautiful UI** - Glassmorphism design with smooth animations
-  **Quick Access** - Direct links to open projects in new tabs
-  **Easy Navigation** - Filter by projects or lectures
-  **Live Previews** - See projects running in real-time with iframes

### How to Use:
1. Open `live-view.html` in your browser
2. Browse all projects in the grid view
3. Use filters to show only Projects or Lectures
4. Type a name (e.g., "Lecture 8" or "Project 1") and press **Enter** for focused view
5. Click "Back to All Projects" to return to grid view

##  Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- No additional setup required!

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/anubhavy-05/Mini_Web_project.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Mini_Web_project
   ```

3. Open `live-view.html` in your browser to access all projects

### Alternative - Individual Projects
You can also open any project directly:
```bash
# Example: Open Project 1 (Calculator)
cd Project1
# Open index.html in your browser
```

##  Usage Examples

### View All Projects
Simply open `live-view.html` and browse the grid of all projects.

### View Specific Project
1. Type the project name in the search box (e.g., "Project 1")
2. Press **Enter**
3. View the project in a larger, focused iframe
4. Click "Open in New Tab" for fullscreen view

### Filter Projects
- Click **"All"** to show everything
- Click **"Projects"** to show only projects
- Click **"Lectures"** to show only lectures
